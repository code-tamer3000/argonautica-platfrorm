# Аргонавтика Web UI Kit

## Что это
Полный кинематографичный сайт-манифест Аргонавтики — единый скролл, точка сбора внимания для
трафика из Instagram. Создаёт напряжение, фильтрует по смелости, ведёт в Манифест и собирает
заявку на Экспедицию (единая точка конверсии).

## Структура файлов

```
index.html      ← Точка входа (грузит Shared → Sections → Sections2 → Sections3 → App)
Shared.jsx      ← Токены, glyph'ы, утилиты (StarSpark, WordMark, MovementGlyph,
                  FadeSection, SecLabel, Hairline, MeanderRule, useParallax, scrollTo, MEDIA)
Sections.jsx    ← Header · Hero (Порог) · О чём
Sections2.jsx   ← Манифест (книжный режим) · Карта (тизер напряжения)
Sections3.jsx   ← Экспедиция (заявка, регистр A) · Footer
App.jsx         ← Root: собирает секции, отслеживает activeSection
media/          ← Атмосферные изображения по ролям-регистрам (см. ниже)
```

## Секции (дуга превращения)

| ID | Секция | Регистр | Описание |

## Компоненты из Shared

- `C`, `MEDIA` — токены цветов и пути к изображениям
- `StarSpark` — 4-лучевая звезда-знак
- `WordMark` — словесный знак (принимает `text`, по умолчанию «АРГОНАВТИКА»)
- `MovementGlyph` — глифы трёх движений: `yav` (прицел), `nav` (сфера+спуск), `prav` (вспышка)
- `FadeSection` — fade+translateY при входе в viewport; уважает `prefers-reduced-motion`; safety-fallback
- `SecLabel` — Onest uppercase метка секции (num + text)
- `Hairline` / `MeanderRule` — золотые разделители (волосяной / греческий меандр, регистр A)
- `useParallax(speed)` — медленный вертикальный параллакс (fade + лёгкий сдвиг)
- `scrollTo(id, offset)` — плавный скролл к секции

## Ощущение
Медленно, веско, кинематографично. Fade + лёгкий параллакс, без прыгучей анимации.
Тёмные грунты (#0B100E / #000000 чередуются паузами), золотой свет, кровь (#8E2018) дозированно,
живая/резная фактура, щедрая тишина. Напряжение нарастает к заявке. Главный закон — живое против
неживого: стерильность = провал.
